Setting the lattice:

```
$ lattice context set --lattice my-lattice
lattice: my-lattice
system:
```

Setting the system:

```
$ lattice context set --system my-system
lattice: my-lattice
system: my-system
```

Setting both the lattice and the system:

```
$ lattice context set --lattice my-lattice-2 --system my-system-2
lattice: my-lattice-2
system: my-system-2
```
